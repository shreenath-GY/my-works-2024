Subject: Formating elements or Formating tags

1). Bold tag: "b" tag is used to make the text as bold without any importance

For Example:

* This is the normal text

* This is the bold text

2). Strong tag: "strong" text which defines the text as bold and browser will understand the text will importance

For Example:

* This is the normal text

* This is the strong text

3). i tag: "i" tag is used to display the text in italic format

For Example:

* This is the normal text

* This is the italic text

4). emphasized tag: "em" tag is a inline tag in which text we write anything inside the em tag that will display the text in italic

For Example:

* This is the normal text

* This is the emphasized text

5). small tag: "small" tag is used to, text we write anything inside the small tag it define the small text than the normal text in our website

For Example:

* This is the normal text

* This is the small text

6). mark tag: "mark" tag is used to define the highlighted text in our website

For Example:

* This is the normal text

* This is the marked text

7).underline tag: "u" tag is used to underline the text.

For Example:

* This is the normal text

* This is the underline text

8). delete tag: "del" tag it defines the deleted text in our website.The text inside the del has been deleted from the document. So the browser will usually strike a line through that deleted text

For Example:

* This is the normal text

* This is the deleted text
